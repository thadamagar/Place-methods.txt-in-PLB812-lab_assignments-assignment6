GO term enrichment with topGOMethods for Download & Format GO terms1. Annotation file ÒBdistachyonABR4_364_v1.ABR4.1.annotation_info.txtÓ has been downloaded from Phytozomoe .2. This annotation file has been changed to the Go term format to match gene names in previous DESeq2 results. do  terms=$(grep $line BdistachyonABR4_364_v1.ABR4.1.annotation_info.txt | cut -f10 | tr ',' '\n' | sort | uniq | sed '/^$/d' | tr '\n' ',' | sed 's/\,$//'); echo "$line $terms" | tr ' ' '\t' | sed 's/\,/\,\ /g' >> Bdistachyon_topGO.tsv; doneRun topGO1. topGO package has been installed and loaded.2. deseq2 result has been loaded res <- read.table("deseq2_results.tsv",header=TRUE,sep="\t")3. GO term file has been loaded goTerms <- readMappings(file="Athaliana_topGO.tsv")4. there was mismatch in name of gene id in DESeq result (Brdisv1ABR41000008m.g ) and annotation (Brdisv1ABR41000001m.g.v1.ABR4.1) which was matched using sub commandres$id <- sub('.v1.ABR4.1', '', res$id)5. Higher expressed genes formatted using upGenes <- factor(as.integer(res$id %in% res[res$padj < 0.05 & res$log2FC >= 1,]$id))names(upGenes) <- res$id6. Lower expressed genes formatted using 7. downGenes <- factor(as.integer(res$id %in% res[res$padj < 0.05 & res$log2FC <= 1,]$id))names(upGenes) <- res$id8. GO term Enrichment table has been created for biological process, molecular function and cellular componentBP <- new("topGOdata",description="Biological Process",ontology="BP",             allGenes=upGenes,annot=annFUN.gene2GO,nodeSize=3,gene2GO=goTerms)FisherBP <- runTest(BP,algorithm="parentchild",statistic="fisher")BPgenTable <- GenTable(BP,Fisher=FisherBP,ranksOf="Fisher",                          topNodes=length(score(FisherBP)))BPgenTable$Fisher<-gsub("< ","",BPgenTable$Fisher)BPgenTable$fdr <- p.adjust(BPgenTable$Fisher,method="fdr")9. .tsv file for each condition has been created higher and lower expressed gene for all biological process, molecular function and cellular componenteg: write_tsv(BPgenTable, file = "higher_expressed_BP.tsv")
